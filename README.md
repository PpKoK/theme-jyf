## 🚀 快速开始

### 环境要求

- Halo 版本 >= 2.22.8
## ⚙️ 主题配置

主题提供了丰富的配置选项，可在 Halo 后台「外观」->「主题」->「主题设置」中配置：

- **基础设置** - Logo、站点标题、描述、热力图等
- **样式设置** - 配色、主色调、背景萤火虫等
- **页脚设置** - 版权信息、ICP 备案等
- **关于页面** - 个人信息、标签、联系方式等
- **TODO 管理** - 任务列表管理

## 📄 自定义模板
